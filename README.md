## Dev Toolkit

A web-based developer toolset that provides various utilities for common development tasks, built with modern web technologies.

### Features

- [ ] JSON
  - [ ] Import and export: JSON string / CSV / YAML / Markdown Table / HTML Query Param / Java Map & List log
  - [ ] Formatter / Minifier / Sorter
  - [ ] Key style converter: camelCase / snake_case ...
  - [ ] JSONSchema validator / generator
  - [ ] JSON Path
  - [ ] JSON Repair
  - [ ] (optional) Parse possible json string in value
- [ ] Encode / Decode
  - [ ] Base64
  - [ ] Hex
  - [ ] URL
  - [ ] Unicode
  - [ ] HTML
- [ ] Crypto
  - [ ] AES/DES
  - [ ] RSA
  - [ ] MD5
- [ ] Regexp Tester
- [ ] Color util
  - [ ] RGB
  - [ ] HEX
  - [ ] HSL
  - [ ] HWB
  - [ ] OKLCH
  - [ ] Color Picker
- [ ] QRCode
- [ ] UUID
- [ ] Text diff compare
- [ ] SQL
  - [ ] Mybatis SQL Param
  - [ ] JSON Array / CSV / Excel / Markdown table / Insert SQL mutual converter
  - [ ] SQL to Java / TS
