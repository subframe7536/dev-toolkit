# Product Overview

Dev Toolkit is a privacy-first web-based developer toolset providing 14+ essential utilities for common development tasks. All processing happens client-side with no server communication, enabling offline functionality after initial load.

## Tool Categories

- **JSON Tools**: Formatter, converter (key case conversion), schema generator
- **Encoding Tools**: Base64, Hex, HTML entities, Unicode, URL encoding/decoding
- **Utilities**: Color converter, date/time formatter, image processor, SQL parameter filler, table editor, UUID generator

## Key Features

- Privacy-first: All computation happens in the browser
- Offline-capable: PWA with service worker support
- Type-safe routing: File-based routing with auto-generated types
- Responsive UI: Built with accessible Kobalte components
